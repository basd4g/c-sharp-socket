# C#-Socket-Communication

## Run

### server program

```sh
$ mcs Program.cs
$ mono Program.exe
```

## Environment

on macOS 10.14.6

```sh
# Install development platform based on the .NET Framework
brew install mono

# compile
$ mcs hoge.cs

# Execute
$ mono hoge.exe
```

reference: [MacでC#のコードをコンパイルして実行する - Qiita](https://qiita.com/matsuda_sinsuke/items/76068f4c396887459803)
